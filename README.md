# Шаблон LaTeX для журнала "Вестник МГУ. Серия 13. Востоковедение"

## Структура проекта

- **vestnik.sty** — пакет со всеми настройками форматирования и макросами
- **article.tex** — основной документ для написания статьи (с ручным списком литературы)
- **article-bibfile.tex** — пример использования .bib файла с biblatex
- **references.bib** — пример библиографической базы данных
- **vestnik.tex** — полный монолитный шаблон (устаревший, для справки)

## Быстрый старт

1. Откройте файл `article.tex`
2. Заполните метаданные статьи (название, автор, аннотация и т.д.)
3. Напишите текст статьи в соответствующих разделах
4. Скомпилируйте документ:
   ```bash
   pdflatex article.tex
   ```

## Использование

### Основной документ

В файле `article.tex` вам нужно только:
- Подключить пакет `vestnik` через `\usepackage{vestnik}`
- Заполнить содержимое статьи

Все технические настройки (шрифты, поля, интервалы, макросы) находятся в файле `vestnik.sty` и подключаются автоматически.

### Доступные макросы

#### Русская версия:
- `\udc{код}` — УДК
- `\articletitleru{название}` — заголовок статьи
- `\authorru{автор}` — автор(ы)
- `\institution{организация}` — организация и адрес
- `\abstractru{текст}` — аннотация (200-300 слов)
- `\keywordsru{слова}` — ключевые слова (5-7)
- `\funding{текст}` — финансирование (опционально)
- `\forcitation{текст}` — для цитирования

#### Английская версия:
- `\articletitleen{title}` — заголовок статьи
- `\authoren{author}` — автор(ы)
- `\institutionen{institution}` — организация и адрес
- `\aboutauthor{info}` — информация об авторах
- `\abstracten{text}` — abstract (200-300 words)
- `\keywordsen{words}` — key words (5-7)
- `\fundingen{text}` — funding (опционально)
- `\forcitationen{text}` — for citation

#### Библиография:
- `\bibliographyru` — начало раздела "Список литературы"
- `\bibliosectioncyrillic` — подраздел кириллицы
- `\bibliosectionlatin` — подраздел латиницы
- `\bibliosectionoriental` — подраздел иероглифики
- `\referencesen` — начало раздела "References"
- `\begin{bibliolist}...\end{bibliolist}` — окружение для ненумерованного списка источников (**используйте для журнала**)
- `\footnote{текст}` — подстрочное примечание для цитирования в тексте (**рекомендуется для журнала**)

**Устаревшие/не соответствующие требованиям журнала:**
- ~~`\begin{bibliolistnum}...\end{bibliolistnum}`~~ — нумерованный список (НЕ используйте для журнала)
- ~~`\bibref{}`~~ — числовые ссылки (НЕ используйте для журнала, используйте `\footnote{}`)

## Работа со списком литературы

⚠️ **ВАЖНО:** Согласно требованиям журнала "Вестник МГУ. Серия 13. Востоковедение":

1. **Ссылки в тексте** оформляются как **подстрочные примечания** (сноски) с полным библиографическим описанием
2. **Список литературы** в конце статьи идет **БЕЗ номеров**, в алфавитном порядке (кириллица → латиница → иероглифика)

Шаблон поддерживает два способа работы:
1. **Ручное создание** (файл `article.tex`) — рекомендуется для журнала
2. **Автоматическая генерация из .bib файла** (файл `article-bibfile.tex`) — для справки

---

### Способ 1: Ручное создание (рекомендуется для журнала)

#### Цитирование через подстрочные примечания

**Правильный подход согласно требованиям журнала:**

```latex
% В тексте статьи используйте сноски
\section{Введение}
Согласно исследованиям\footnote{Иванов И.И. Название книги. М.: Издательство, 2020. С. 45.}, данная проблема...
Как отмечает автор\footnote{Петров П.П. Название статьи // Название журнала. 2021. № 1. С. 15.}, важным аспектом является...
```

При повторной ссылке на тот же источник можно использовать краткую форму:
```latex
...как было показано ранее\footnote{Иванов И.И. Указ. соч. С. 67.}
```

#### Список литературы в конце статьи

**БЕЗ номеров, в алфавитном порядке:**

```latex
\bibliographyru

% Раздел на русском языке (кириллица) - в алфавитном порядке
\bibliosectioncyrillic
\begin{bibliolist}
\item Иванов И.И. Название книги. М.: Издательство, 2020. 200 с.
\item Петров П.П. Название статьи // Название журнала. 2021. № 1. С. 10--20.
\end{bibliolist}

% Раздел на западных языках (латиница) - в алфавитном порядке
\bibliosectionlatin
\begin{bibliolist}
\item Johnson M. Article Title // Journal Name. 2020. Vol. 10. No. 2. P. 50--60.
\item Smith J. Book Title. New York: Publisher, 2019. 300 p.
\end{bibliolist}
```

#### Оформление References (английская версия)

**Требования к References:**
- Сохранить порядок как в "Списке литературы"
- Названия книг, статей, сборников выделяются курсивом (`\textit{}`)
- Для русских/восточных источников: транслитерация + перевод [в скобках]
- В конце каждой ссылки указывается язык: `(In Russ.)`, `(In Arab.)` и т.д.

```latex
\referencesen

\begin{bibliolist}
\item Ivanov I.I. \textit{Nazvanie knigi} [Book Title]. Moscow, Izdatel'stvo Publ., 2020, 200 p. (In Russ.)
\item Petrov P.P. Nazvanie stat'i [Article Title]. \textit{Nazvanie zhurnala} [Journal Name], 2021, no. 1, pp. 10--20. (In Russ.)
\item Johnson M. Article Title. \textit{Journal Name}, 2020, vol. 10, no. 2, pp. 50--60.
\item Smith J. \textit{Book Title}. New York, Publisher, 2019, 300 p.
\end{bibliolist}
```

### Способ 2: Автоматическая генерация из .bib файла

Если у вас большое количество источников или вы хотите автоматизировать библиографию, используйте файл `article-bibfile.tex` с пакетом **biblatex**.

#### Создание .bib файла

Создайте файл `references.bib` с библиографическими записями:

```bibtex
@book{ivanov2020,
    author    = {Иванов, И. И.},
    title     = {Название книги},
    publisher = {Издательство},
    address   = {М.},
    year      = {2020},
    pages     = {200},
    language  = {russian}
}

@article{smith2019,
    author   = {Smith, John},
    title    = {Article Title},
    journal  = {Journal Name},
    year     = {2019},
    volume   = {10},
    pages    = {50--60},
    language = {english}
}
```

#### Использование в тексте

```latex
% В преамбуле
\usepackage[backend=biber, style=numeric]{biblatex}
\addbibresource{references.bib}

% В тексте статьи
Согласно исследованиям \cite{ivanov2020}, данная проблема...
Как отмечает автор \cite[с. 45]{smith2019}, важным является...

% В конце статьи
\printbibliography[title={\centering\textbf{Список литературы}}]
```

#### Компиляция с biblatex

Для корректной работы выполните последовательность команд:

```bash
pdflatex article-bibfile.tex
biber article-bibfile
pdflatex article-bibfile.tex
pdflatex article-bibfile.tex
```

Или используйте `latexmk` для автоматической компиляции:

```bash
latexmk -pdf article-bibfile.tex
```

#### Преимущества .bib файла

✅ Автоматическая нумерация источников
✅ Автоматическая сортировка по алфавиту
✅ Переиспользование библиографии в разных статьях
✅ Автоматическое форматирование ссылок
✅ Включение только цитируемых источников

**Примечание:** Файл `references.bib` содержит примеры для разных типов источников: книги, статьи, диссертации, онлайн-ресурсы.

## Требования к форматированию

Все требования журнала автоматически соблюдаются:
- ✅ Шрифт Times New Roman, 12pt
- ✅ Межстрочный интервал 1.5
- ✅ Поля 1 дюйм (2.54 см) со всех сторон
- ✅ Абзацный отступ 1.25 см
- ✅ Сноски 10pt, интервал 1.0
- ✅ Нумерация всех страниц
- ✅ Поддержка русского языка

## Компиляция

Для компиляции документа используйте команду:
```bash
pdflatex article.tex
```

При необходимости повторите компиляцию дважды для корректного отображения ссылок.

## Примечания

- Файл `vestnik.tex` содержит старую версию шаблона с встроенными настройками. Вы можете использовать его для справки, но рекомендуется работать с `article.tex`.
- Все комментарии в файлах на русском языке для удобства.
- Шаблон предназначен для текстовых статей без сложных математических формул и таблиц.

## Помощь

При возникновении проблем с компиляцией убедитесь, что:
1. Файлы `article.tex` и `vestnik.sty` находятся в одной папке
2. У вас установлен полный дистрибутив LaTeX (TeX Live или MiKTeX)
3. Установлены необходимые пакеты (обычно устанавливаются автоматически)
